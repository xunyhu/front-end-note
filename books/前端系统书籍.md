# 前端系统书籍

## 🧱 第一阶段：基础夯实（HTML / CSS / JS核心）

目标：补齐底层理解，而不是会用

必读（优先级最高）

- 《JavaScript高级程序设计（红宝书）》
  👉 前端圣经，系统讲清变量、作用域、原型、异步等
- 《你不知道的JavaScript（上中）》
  👉 深入理解 JS 本质（闭包 / this / 作用域）

## ⚙️ 第二阶段：进阶（浏览器 + 工程能力）

目标：从“会写页面” → “懂前端”

- 《高性能JavaScript》
  👉 性能优化思维
- 《WebKit技术内幕》
  👉 浏览器原理（加分项）
- 《深入理解ES6》
  👉 现代 JS 核心
- 《前端架构：从入门到微前端》
  👉 架构视角

## ⚛️ 第三阶段：框架（React / Vue / TS）

- 《深入浅出React》
- 《React入门经典》
  👉 系统理解组件、状态、路由
- 《Vue.js设计与实现》
- 《深入理解TypeScript》

## 第四阶段：工程 & 架构

- 《设计数据密集型应用》
- 《前端工程化体系设计》
- 《微前端设计与实现》
