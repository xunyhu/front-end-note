# JS概览与类型

## 1️⃣ 数据类型

### 概念

JavaScript 的数据类型分为两类：

- 原始类型：Number、String、Boolean、Null、Undefined、Symbol、BigInt
- 对象类型：Object、Array、Function 等

### 原理

- 原始类型按值存储，赋值和传参复制值
- 对象类型按引用存储，赋值和传参是引用传递

### 用途/场景

数据类型判断、类型转换、理解值与引用差异

### 示例代码

```javascript
let num = 10;
let str = "hello";
let obj = { a: 1 };
let copyObj = obj;
copyObj.a = 2;
console.log(obj.a); // 2
```

### 总结

JS 有7种原始类型和3种对象类型，原始类型按值传递，对象类型按引用传递。

## 2️⃣ 类型转换

### 概念

JS会在需要时自动或手动进行类型转换

### 原理

- 数字运算：字符串转换为数字
- 字符串拼接：数字转换为字符串
- 布尔上下文：falsy值包括0, "", null, undefined, NaN, false; 空对象为true;

### 用途/场景

避免隐式转换带来的逻辑错误

### 示例代码

```
"5" - 2; // 3
"5" + 2; // "52"
Boolean(""); // false
```

### 总结

- +运算符触发字符串拼接，-运算符触发数值转换；布尔上下文空字符串、0、null、undefined 为 false。

## 3️⃣ 相等比较

### 概念

== 会进行类型转换再比较，=== 不进行类型转换

### 原理

- == 转换类型后比较
- === 直接比较类型和值

### 用途/场景

防止隐式转换导致bug。建议实际开发和面试中使用 ===，避免隐式类型转换。

### 示例代码

```js
0 == "0"; // true
0 === "0"; // false
null == undefined; // true
```
