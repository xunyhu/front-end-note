# 异步与事件循环

## Event Loop

### 概念

JS执行模型，单线程 + 异步队列

### 示例

```js
console.log(1);
setTimeout(() => console.log(2), 0);
console.log(3);
// 输出: 1 3 2
```

宏任务队列（setTimeout/setInterval）和微任务队列（Promise.then），微任务优先执行。

## Promise

### 概念

异步编程方案，链式调用。

### 示例

```js
new Promise((resolve) => resolve(1)).then((res) => console.log(res)); // 1
```

Promise 是状态机，有 pending/resolved/rejected 三种状态，then 返回新的 Promise。

## async/await

### 概念

语法糖封装 Promise，使异步看起来同步。

### 示例

```js
async function test() {
  const res = await Promise.resolve(1);
  console.log(res);
}
test();
```

await 暂停 async 函数，异步操作顺序执行，更易读。
