# 作用域与闭包

## 作用域

### 概念

变量和函数可访问的范围

### 原理

- JS是词法作用域
- 内层作用域可访问外层变量
- 执行上下文 = 变量环境 + 词法环境

### 用途/场景

闭包理解、变量隔离、防止全局污染

### 示例代码

```js
function outer() {
  var a = 1;
  function inner() {
    console.log(a);
  }
  inner();
}
outer();
```

### 总结

作用域链是函数查找变量的机制，闭包是作用域链被保留下来的典型例子。

## 闭包

### 概念

函数访问外部作用域变量形成的持久引用

### 原理

- 内层函数引用外层变量
- 即使外层函数执行完成，变量依然存在

### 用途/场景

数据私有化、函数工厂、事件回调

### 示例代码

```js
function makeCounter() {
  let count = 0;
  return function () {
    count++;
    return count;
  };
}
const counter = makeCounter();
console.log(counter()); // 1
console.log(counter()); // 2
```

### 总结

闭包本质是词法作用域被保留下来，形成变量持久引用。
