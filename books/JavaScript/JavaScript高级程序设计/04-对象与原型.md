# 对象与原型

## 原型链

### 概念

对象通过 proto 链访问父对象属性。

```js
function Person() {
  this.name = "Tom";
}
Person.prototype.say = function () {
  console.log(this.name);
};
var p = new Person();
p.say(); // Tom
```

## 对象创建方式

### 概念

字面量、构造函数、Object.create()

### 示例

```js
var obj1 = { a: 1 };
var obj2 = new Object();
var obj3 = Object.create(obj1);
```

Object.create可以指定原型对象，比构造函数更灵活。

## ES6 Class

### 概念

class 是语法糖，背后是原型机制。

### 示例

```js
class Person {
  constructor(name) {
    this.name = name;
  }
  say() {
    console.log(this.name);
  }
}
```

class 只是语法糖，底层依旧是函数+原型。
