# 函数

## this

### 概念

执行上下文中的this指向，取决于调用方式。

### 示例

```js
function test() {
  console.log(this);
}
test();
```

this 是函数调用时动态绑定的，箭头函数没有自己的 this，会继承外层作用域。

## call / apply / bind

### 概念

改变函数调用时的 this 指向。

### 示例

```js
function say() {
  console.log(this.name);
}
say.call({ name: "Tom" }); // Tom
```

## 函数声明 vs 函数表达式

### 概念

函数声明提升，函数表达式不提升。

### 示例

```js
foo(); // 可以
function foo() {
  console.log(1);
}

bar(); // Error
var bar = function () {
  console.log(2);
};
```

函数声明会在编译阶段提升，函数表达式不会。
