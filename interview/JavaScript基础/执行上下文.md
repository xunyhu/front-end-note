# 🧠 JavaScript 执行上下文 + 作用域 + 闭包

---

## 📌 使用说明

本练习用于强化以下核心知识点：

- 执行上下文（Execution Context）
- 变量提升（Hoisting）
- 作用域链（Scope Chain）
- 闭包（Closure）

建议练习方式：

1. **独立思考输出结果**
2. **写出执行过程（创建阶段 + 执行阶段）**
3. **用面试语言复述原因**

---

# 🧩 练习题

---

## ① 变量提升（基础）

```js
console.log(a);
var a = 1;
```

👉 输出是什么？为什么？

---

## ② 函数 vs 变量提升（经典）

```js
console.log(foo);

function foo() {}
var foo = 1;
```

👉 输出是什么？为什么？

---

## ③ 函数内部变量提升（必考🔥）

```js
var a = 1;

function foo() {
  console.log(a);
  var a = 2;
}

foo();
```

👉 输出是什么？为什么？

---

## ④ 作用域链查找（核心）

```js
var a = 1;

function foo() {
  function bar() {
    console.log(a);
  }
  bar();
}

foo();
```

👉 输出是什么？变量是如何查找的？

---

## ⑤ 闭包基础（高频🔥）

```js
function foo() {
  var a = 1;
  return function () {
    console.log(a);
  };
}

var fn = foo();
fn();
```

👉 输出是什么？为什么 foo 执行完还能访问 a？

---

## ⑥ 闭包 + 变量变化（进阶）

```js
function foo() {
  var a = 1;
  return function () {
    a++;
    console.log(a);
  };
}

var fn = foo();
fn();
fn();
```

👉 每次输出是什么？为什么？

---

## ⑦ 多个闭包是否共享变量（重点🔥）

```js
function foo() {
  var a = 1;
  return function () {
    console.log(a);
  };
}

var fn1 = foo();
var fn2 = foo();

fn1();
fn2();
```

👉 输出是什么？fn1 和 fn2 是否共享 a？

---

## ⑧ for + 闭包（面试高频）

```js
for (var i = 0; i < 3; i++) {
  setTimeout(function () {
    console.log(i);
  }, 0);
}
```

👉 输出是什么？为什么？

---

## ⑨ 改造上题（let）

```js
for (let i = 0; i < 3; i++) {
  setTimeout(function () {
    console.log(i);
  }, 0);
}
```

👉 输出是什么？为什么？

---

## ⑩ 综合题（王炸🔥）

```js
var a = 1;

function foo() {
  var a = 2;

  return function () {
    console.log(a);
  };
}

function bar(fn) {
  var a = 3;
  fn();
}

var fn = foo();
bar(fn);
```

👉 输出是什么？为什么？

# 🎯 面试表达模板（建议背）

你可以用这种结构回答：

> 1. 当前代码会创建执行上下文
> 2. 在创建阶段发生变量提升
> 3. 执行阶段逐行执行代码
> 4. 变量通过作用域链向上查找
> 5. 如果涉及闭包，会保留外部作用域引用
